# BestProf — Корпоративный сайт

Корпоративный сайт ТОО «BestProf» — казахстанского производителя алюминиевых и пластиковых профильных систем.

## Структура проекта

```
bestprof/
├── index.html              # Главная страница
├── css/
│   ├── variables.css       # CSS-переменные, сброс стилей, типографика
│   ├── header.css          # Верхняя панель + навигация
│   ├── hero.css            # Hero-секция с SVG профилем
│   ├── catalog.css         # Каталог продукции + карточки товаров
│   ├── sections.css        # О компании, услуги, партнёры, преимущества
│   ├── footer.css          # CTA-форма + подвал сайта
│   ├── animations.css      # Scroll-reveal анимации
│   └── responsive.css      # Адаптив (1024px, 768px)
├── js/
│   └── main.js             # Модульный JS (навигация, табы, форма, маска)
├── img/
│   └── favicon.svg         # Фавикон
└── README.md
```

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — CSS Grid, Flexbox, CSS-переменные, анимации
- **JavaScript** — Vanilla JS, Intersection Observer API
- **Шрифты** — Plus Jakarta Sans (заголовки), DM Sans (текст)
- **SVG** — иконки, фавикон, анимированный профиль в hero
